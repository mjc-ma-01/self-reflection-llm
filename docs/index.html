Reflector

Reflector trains a model to inspect its own answer before finalizing, separating risky initial impulses from a safe and useful final response.

Read arXiv GitHub Model Try the demo

A reproducible path from reflection data to evaluated checkpoints.

The repository keeps data loading, SFT, GDPO reinforcement learning, and benchmark export under consistent interfaces so experiments can be re-run instead of reconstructed.

Unified pattern data

Training consumes only harmful and general pattern files, while public benchmark reports expose the General Benchmark and external suites.

GDPO preserved

The reinforcement learning entrypoint keeps the original GDPO reward and advantage flow, with only data formatting adapted to the new schema.

Paper-ready results

Evaluation writes structured JSON, CSV, and Markdown reports for direct inspection and reuse.

Start locally.

Clone, install dependencies, prepare data, train, and evaluate from the same shell.

git clone https://github.com/mjc-ma-01/self-reflection-llm.git
cd self-reflection-llm

conda create -n reflector python=3.11 -y
conda activate reflector
pip install -r requirements.txt

export HF_HOME=$PWD/.cache/huggingface
export HUGGINGFACE_HUB_CACHE=$HF_HOME/hub

bash scripts/sft/prepare.sh
MODEL_PATH=meta-llama/Meta-Llama-3-8B-Instruct bash scripts/sft/train.sh
bash scripts/sft/merge_lora.sh
NUM_SAMPLES=50 bash scripts/sft/validate.sh
